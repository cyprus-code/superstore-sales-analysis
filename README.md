Superstore Sales Analysis



**Project Overview**

End-to-end sales data analysis project using R and Power BI.



**Tools Used**

\- R (tidyverse, lubridate, janitor)

\- Power BI Desktop



**Key Findings**

1\. West region leads with 14.9% profit margin

2\. Central region over-discounting — 21.4% of orders exceed 30% discount

3\. Furniture category has only 2.49% margin vs Technology at 17.4%

4\. Discounts above 30% result in negative profit margins

\-  data — cleaned dataset
